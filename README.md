# list_task

想让你的男朋友像玩游戏一样，完成你交代的任务么？那就试试这个程序吧。

---

## 做这个小程序的初衷
平时拿着一堆菜、在超市里对照清单买东西时，手忙脚乱地操作手机真的很不方便；
加上刷到很多博主分享，情侣之间因为任务交代不清晰、理解有偏差，很容易产生不必要的误会。

所以我做了这个像游戏任务列表一样的小程序：
把要交代的事情一条条写清楚，谁完成了、还剩哪些没做，都一目了然，
就像游戏里的任务面板一样，目标明确，再也不会因为“没说清楚”而闹小别扭啦。

---

## 不只是买菜，这些场景也能用
它不只是个购物清单，更像一个「任务传递神器」，你可以用它：
- 🛒 买菜购物：把要带的东西列成任务，发给一起逛超市的朋友/伴侣，再也不会漏买
- 🏠 家务分工：把扫地、洗碗、倒垃圾列成任务，发给家人，谁完成谁打勾，清清楚楚
- 📝 工作/学习协作：把待办事项发给同事/同学，进度一目了然
- 🎁 给朋友的小惊喜：把准备礼物、布置场景的步骤做成任务，发给朋友一起完成，仪式感拉满

只要你想把事情说清楚，都可以用这种“任务清单”的方式，发给微信好友一起协作。

## 未来的小想法
希望以后能慢慢把它做得更实用、更贴心：
- 📋 加入**任务模板库**：比如“周末大扫除”“出游准备清单”，一键生成常用任务，不用每次都从头写
- 🍳 加入**菜谱联动功能**：选好一道菜，自动生成对应的采购清单和步骤任务，做饭、买菜都不慌
- ⏰ 增加**任务提醒**：快到截止时间时自动提醒，再也不怕忘了要做的事
- 🎨 丰富**游戏化体验**：比如完成任务解锁小成就、累计积分，让完成任务变得更有成就感

如果你也觉得这些想法有意思，或者有更好的点子，欢迎一起交流完善！
